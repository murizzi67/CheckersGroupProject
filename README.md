# CheckersGroupProject
Applicazione WPF del gioco "dama" con implementazione di un algoritmo ricorsivo con backtracking 
## CARATTERISTICHE PRINCIPALI DELL'APPLICAZIONE 
- Schermata di menu
- Scacchiera formata da una griglia riempita da blocchi singoli esterni (bianchi e neri) richiamati nella finestra principale
- Pedine create con dei blocchi esterni, richiamate e sovrapposte alle caselle della scacchiera
  - Al raggiungimento dell'ultima riga della scacchiera la pedina dama, permettendole di muoversi anche all'indietro
- Algoritmo di gioco indiperndente che fa da avversario
## IMPLEMENTAZIONI FUTUTRE
- Animazione di vittoria e sconfitta
- Possibilita' di mangiare piu' pedine in una singola mossa nel caso esse siano posizionate nei punti adatti
